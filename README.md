# MLOps: Sentiment Pipeline (DVC + MLflow)

## Project layout
```text
.
|-- data/
|   |-- dataset.csv(.dvc)      # raw dataset tracked by DVC
|   |-- prepared/              # generated by prepare stage (DVC out)
|   `-- models/                # generated by train stage (DVC out)
|-- src/
|   |-- stages/
|   |   |-- prepare.py         # stage 1: clean + split
|   |   `-- train.py           # stage 2: train + metrics + MLflow logging
|   |-- experiments/
|   |   `-- train_mlflow.py    # standalone multi-run experiment script
|   |-- __init__.py
|   |-- stages/__init__.py
|   `-- experiments/__init__.py
|-- dvc.yaml                   # pipeline definition
|-- dvc.lock                   # frozen stage inputs/outputs hashes
|-- notebooks/
|   `-- analysis.ipynb         # exploratory notebook
|-- pyproject.toml
`-- uv.lock
```

## Install dependencies
```bash
uv sync
```

## Run DVC pipeline
```bash
uv run dvc repro
```

## Run standalone experiments
```bash
uv run python src/experiments/train_mlflow.py
```

## Open MLflow UI
```bash
uv run mlflow ui --backend-store-uri ./mlruns
```

Open `http://127.0.0.1:5000` in browser.
